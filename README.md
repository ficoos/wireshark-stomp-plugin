Wireshark STOMP dissector plugin
================================

## Installation

#### Linux
Copy `stomp.lua` to `~/.wireshark/plugins/`

#### MacOS
Copy `stomp.lua` to `~/.config/wireshark/plugins/`

#### Windows
Copy `stomp.lua` to `%APPDATA%\Wireshark\plugins`
